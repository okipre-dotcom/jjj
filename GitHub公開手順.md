# FP1級トレーナーをiPadで「1タップ起動・完全オフライン」にする手順

所要時間: 約10分(初回のみ)。以後はホーム画面のアイコンをタップするだけです。

## 1. GitHubアカウントを作る(持っていれば飛ばす)
1. ブラウザで https://github.com/signup を開く
2. メールアドレス・パスワード・ユーザー名(半角英数。例: takeru-fp1)を登録
   ※ユーザー名は公開URLの一部になります

## 2. リポジトリ(置き場所)を作る
1. ログイン後、右上の「+」→「New repository」
2. Repository name に「 fp1 」と入力
3. 「Public」を選択(そのまま)→「Create repository」
   ※Publicですが、URLを知らない人が見つけることはまずありません。気になる場合は後述の注意参照

## 3. ファイルをアップロードする
1. 作ったリポジトリの画面で「uploading an existing file」のリンクをタップ
   (見当たらなければ「Add file」→「Upload files」)
2. 「choose your files」→ ファイルアプリからこのフォルダ(GitHub公開用)の中身を全部選ぶ
   - index.html
   - FP1tra_v5.2_PWA.html (PWA版)
   - FP1tra_v5.2.html (非PWA版・予備)
   - sw.js / manifest.json / icon-192.png / icon-512.png
   ※2.8MBのファイルが2つあるので少し時間がかかります
3. 下の「Commit changes」をタップ

## 4. GitHub Pagesを有効にする
1. リポジトリ上部の「Settings」→ 左メニュー「Pages」
2. Branch を「main」、フォルダ「/(root)」にして「Save」
3. 1〜2分待ってページを再読込すると、上部に公開URLが表示される
   → https://あなたのユーザー名.github.io/fp1/

## 5. iPadのホーム画面に追加(ここが本命)
1. SafariでそのURLを開く(自動でアプリ本体に転送されます)
2. 動作を確認したら、共有ボタン(四角+↑)→「ホーム画面に追加」
3. ホーム画面に「FP1級」アイコンができる → 以後はそれをタップするだけ
   - 全画面表示(ツールバーなし・画面が動く問題も解消)
   - 一度開けばオフラインでも起動(機内モードで試してみてください)
   - 学習履歴も保存されます

Chromeで使いたい場合も同じURLをブックマークすればOKです(オフライン対応・全画面はホーム画面追加が最強です)。

## バージョンアップのとき
1. 新しい「_vX.X_PWA.html」を私(Claude)が作ったら、リポジトリの「Add file」→「Upload files」で
   index.html / sw.js / 新PWAファイル を上げ直す(古いファイルは残してOK)
2. アプリを開き直すと自動で新版に切り替わります(切り替わらない場合は一度閉じて再起動)

## 注意
- 学習履歴は端末内(ブラウザ)に保存されます。GitHubには何も送信されません
- Publicリポジトリが気になる場合: 内容は市販教材由来のため、リポジトリ名を推測されにくいもの(例: fp1-x7k2)にしておくとより安心です
